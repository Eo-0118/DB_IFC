# 서울시 토지피복 및 불투수 면적 데이터 전처리 요약

본 문서는 서울시 토지피복 데이터를 활용하여 연도별, 시군구별 토지 이용 현황 및 불투수 면적을 산출하고 통합하는 전처리 과정을 설명합니다.

## 1. 개요
*   **목적**: 2000년부터 2024년까지의 서울시 25개 구별 토지피복 변화 및 불투수율 데이터를 구축
*   **원본 데이터**: 서울시 토지피복지도 (Level 2/Level 3 분류, 연도별 Shapefile 가공본)
*   **최종 결과물**: `Seoul_Land_info.csv`

## 2. 전처리 단계

### Step 1: 불투수 면적 산출 (`ImperviousData`)
1.  **기준 설정**:
    *   중분류(LV2) 기준으로 불투수/투수 구분
    *   **불투수(Impervious)**: 주거(110), 공업(120), 상업(130), 문화체육(140), 교통(150), 공공(160), 시설재배(230)
    *   **투수(Pervious)**: 농업(210 등), 산림(300), 초지(400), 습지(500), 나지(600), 수역(700), 기타재배(250)
2.  **데이터 추출**: 각 연도별 요약 파일(`*_summary.csv`)에서 위 기준에 따라 면적 집계
3.  **데이터 보간 (Interpolation)**:
    *   데이터가 없는 연도(2001~2006, 2008, 2010~2012, 2015~2017)에 대해 **선형 보간(Linear Interpolation)** 수행
    *   2000년부터 2024년까지 연속적인 시계열 데이터 확보
4.  **결과물**: `Seoul_ImperviousData_2000_2024.csv`

### Step 2: 토지피복 분류 집계 (`Seoul_Land_Cover`)
1.  **분류 기준 (Mixed Level Classification)**:
    *   **시가화 건조지역 (100번대)**: 중분류 단위로 상세 구분 (주거, 공업, 상업, 문화, 교통, 공공)
    *   **비시가화 지역 (200~700번대)**: 대분류 단위로 통합 (농업, 산림, 초지, 습지, 나지, 수역)
2.  **데이터 매핑**:
    *   `mapping_land_cover.py` 스크립트 사용
    *   세분류 코드가 있을 경우 중분류로 정규화 (예: 132 -> 130)
3.  **데이터 보간**:
    *   불투수 데이터와 동일하게 빈 연도에 대해 선형 보간 수행
4.  **결과물**: `Seoul_LandCover_Mapping.csv`

### Step 3: 데이터 통합 및 검증
1.  **병합**:
    *   `merge_land_info.py` 스크립트 사용
    *   `YEAR`, `SIGUNGU_NM`을 기준으로 두 데이터셋 통합
2.  **최종 구조**:
    *   기본 정보: 연도, 시군구명
    *   토지피복 정보: 주거, 공업, 상업, 교통 ... 수역 (면적 ㎡)
    *   불투수 정보: 불투수 면적, 투수 면적, 불투수율(%), 총 면적
3.  **검증**:
    *   각 항목별 면적 합계 vs 총 면적(`TOTAL_AREA_M2`) 비교
    *   소수점 오차(0.01~0.02㎡) 내에서 일치함 확인 완료
